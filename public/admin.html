Claude Desktop Setup

Make sure this environment variable is set (run as Administrator):

[System.Environment]::SetEnvironmentVariable("ANTHROPIC_API_URL", "http://127.0.0.1:8082", "Machine")

Then restart Claude Desktop.

Revert to Anthropic

To go back to real Claude / Sonnet:

[System.Environment]::SetEnvironmentVariable("ANTHROPIC_API_URL", $null, "Machine")

  2. Open Claude Desktop

Adding API Keys

Edit C:\free-claude-desktop\.env and add:

NVIDIA_API_KEY=nvapi-your-key-here
GROQ_API_KEY=gsk_your-key-here
OPENAI_API_KEY=sk-your-key-here

Get keys at: build.nvidia.com | console.groq.com | platform.openai.com

Edit Provider
Leave blank to use the environment variable instead (set in .env file). Key is stored in config.json and never sent to the browser in full.
